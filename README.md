### Requirement

- 基于Java1.8, Maven管理
- 本程序用到了图像识别组件**tesseract**，需要提前在机器上安装，[安装指南](https://github.com/tesseract-ocr/tesseract)
- 需要安装**Chrome**浏览器

### 使用说明

- 运行前，需要在左上角初始化中**配置账户密码**和导入要抓取的**关键词**
- 源码中的`a.txt`为输入文件格式要求，可以**输入多行**
- 如果不想运行源码，可以只下载`executable`目录下的`jar`文件，在配置好以上环境后，直接运行即可
